Question-1: What are the different ways to select an element in the DOM?

For select an element in the DOM, we have 5 way likes: (i)-Selects an element by its id. (ii)- Selects all elements with a specific class. (iii)- Selects all elements with a specific tag. (iv)- Selects the first matching element (CSS selector). (v)- Selects all matching elements (returns a NodeList).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent: innerHTML: Gets or sets the HTML content inside an element (includes tags) (Reads & writes HTML). innerText: Gets or sets only the visible text inside an element (ignores hidden text & tags) (Displays text as plain content). textContent: Gets or sets all text (including hidden text) inside an element.

Question-3: What is event delegation in the DOM?

Event Delegation is when you add one event listener to a parent element instead of adding it to multiple child elements. The event bubbles up from the child to the parent.

Question-4: What is event bubbling in the DOM?

Event Bubbling means that when an event happens on a child element, it first runs on that element, then moves up to its parent, grandparent, and so on until it reaches the most top.

Question-5: How do you create, add, and remove elements using JavaScript?

Create, Add, and Remove elements using JavaScript: Create: To create a new div, p, or any other HTML element that can show your web page dynamically. This new element is like an invisible object that can later be customized with content (like text) or styles.
Add: After creating an element, to add it to the page. You can choose where you want it to appear by selecting a parent element on the page, and then append your new element inside that parent. This makes the newly created element visible on the webpage.
Remove: To remove an element, it can directly delete it from the page. This can be done by calling a method that removes the element from the parent it belongs to, or by removing the element itself if you have a reference to it
